# Forest Fire Early Detection System

## Overview
This project is an AI-powered early detection system for forest fires. It uses computer vision and deep learning to analyze video streams for signs of fire and smoke, providing real-time alerts and risk assessment. The system combines color analysis, thermal overlays, motion tracking, and object detection to identify hazardous situations and assist in wildfire prevention and monitoring.

## How to Fork and Run
1. Fork this repository to your own GitHub account using the "Fork" button.
2. Clone your forked repository:
   ```
   git clone https://github.com/your-username/ForestFireDetection.git
   ```
3. Install the required dependencies (see requirements in the code or use the auto-installer).
4. Run the main script with a video file or webcam:
   ```
   python app.py input.mp4
   python app.py 0  # for webcam
   ```
