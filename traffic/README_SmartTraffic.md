# Smart Traffic System

## Overview
This project is a production-grade AI traffic intelligence and smart city monitoring system. It uses computer vision and deep learning to detect, track, and analyze vehicles and people in real-time from video streams or webcam feeds. The system provides analytics such as vehicle counting, speed estimation, lane analytics, congestion scoring, and more, making it suitable for smart city and traffic management applications.

## How to Fork and Run
1. Fork this repository to your own GitHub account using the "Fork" button.
2. Clone your forked repository:
   ```
   git clone https://github.com/your-username/SmartTraffic.git
   ```
3. Install the required dependencies (see requirements in the code or use the auto-installer).
4. Run the main script with a video file or webcam:
   ```
   python app2.py input.mp4
   python app2.py 0  # for webcam
   ```
